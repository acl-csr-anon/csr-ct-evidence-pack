# scripts

No automation scripts are included for this experiment.

- All model calls (ChatGPT, Gemini, Claude) and file organization steps
  were performed manually.
- The repository is intended as an **evidence pack**, not a reusable
  benchmarking pipeline.

If future work requires re-running the experiments programmatically,
new scripts can be added here, but they are **intentionally omitted**
for the current version.
